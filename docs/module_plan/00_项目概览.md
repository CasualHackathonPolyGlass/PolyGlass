# 项目概览：Polymarket Smart Money Dashboard

**版本**：v0.1 (MVP)
**时间**：2026-01

---

## 目标

基于 Polygon 链上 Polymarket 的真实 `OrderFilled` 事件，构建可验证的数据采集、解码、存储与可视化系统。

---

## MVP 输出

| 输出项 | 说明 |
|--------|------|
| **链上成交记录** | 至少展示 100 条真实数据（含 txHash、block、解码字段） |
| **市场归类** | 交易可归类到具体 Market（通过 tokenId 匹配） |
| **API 查询** | 支持 HTTP API 查询 |
| **Web 可视化** | 提供 Dashboard 界面 |

---

## 技术栈

| 层级 | 技术 |
|------|------|
| **前端** | Next.js + TypeScript + Tailwind + ECharts/Recharts |
| **后端** | Node.js（同仓库 API Routes） |
| **数据库** | SQLite（Prisma/Drizzle） |
| **外部服务** | Polygon RPC + Gamma API |

---
